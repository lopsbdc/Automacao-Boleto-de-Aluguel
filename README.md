# Automação Boleto de Aluguel

Este é um bot de automação para cadastro e pagamento de boletos de aluguel, juntamente ao sistema Oracle.
O bot visa abrir uma relação com os dados para cadastro, e a partir da tela inicial do Oracle, já inicia todos os cadastros em loop.

O bot utiliza reconhecimento de imagens para identificar os botões a serem clicados, e utiliza o teclado para digitar todos os dados. 
Dessa forma, é necessário deixar máquina executando "sozinha", até que seja finalizado o processo
